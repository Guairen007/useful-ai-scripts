# Useful AI Scripts

5 Python scripts for everyday tasks. AI-generated, tested, ready to run.

## Scripts included

| Script | What it does |
|--------|-------------|
| `organize_downloads.py` | Sort your downloads folder by file type |
| `bulk_rename.py` | Rename multiple files at once |
| `csv_to_json.py` | Convert CSV files to JSON format |
| `extract_emails.py` | Pull email addresses out of a text file |
| `merge_pdfs.py` | Combine multiple PDFs into one |

## How to use

Each script has comments explaining what it does. Download and run.
