# Freelancer Project Manager Spreadsheet

A simple Google Sheets / Excel template to manage your freelance projects.

## What's inside
- **Projects sheet**: track name, client, deadline, status, budget
- **Tasks sheet**: assign tasks, set priority, mark as done
- **Clients sheet**: store contact info and notes
- **Invoices sheet**: record invoice number, amount, and payment status

## How to use
1. Download the .xlsx file
2. Open in Excel, Google Sheets, or LibreOffice
3. Replace the example data with your own
4. Stay organized without expensive project management software
